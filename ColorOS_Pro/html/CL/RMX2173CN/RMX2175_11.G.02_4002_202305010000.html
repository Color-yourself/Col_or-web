版本介绍
亲爱的用户，欢迎体验 ColorOS Pro，本版本底层及供应商部分基于官方 realme UI 3.0 F.11 版本、系统部分基于 OPPO K7x F.09 版本制作，为您带来诸多特性，敬请浏览更新日志
注意事项
· 首次刷入此版本时，建议格式化 data 分区刷入 · 若此前刷入此版本，但又刷了其他系统或官方 ROM，再刷此版本也建议格式化 data 分区 · 若有问题、bug或建议，可将意见反馈给作者（反馈入口：设置->关于系统->意见反馈） · 此版本暂未禁用系统更新检测

更新日志
便捷高效
• 新增 Omoji 多种装饰与脸部造型，个性玩法更多样 • 新增隔空手势，在视频与通话场景下，通过隔空手势，可进行接听、静音、滑动等操作（支持部分应用） • 新增亮屏语音唤醒功能
无缝互联
跨屏无缝连接，构建多端互融生态，打造全方位的舒适生活体验 • 新增 PC 跨屏互联 - 支持 PC 与手机间跨屏拖拽传输文件、照片，摆脱有线束缚，快速传输 - 支持 PC 直接编辑手机内的文档，完成后自动存回手机 - 支持 PC 端接收并处理手机聊天软件、短信等通知消息，无需来回切换 - 手机与 PC 剪贴板互通，可直接粘贴剪贴板中的最新内容 - 应用多开功能，支持同时在电脑最多打开三个应用窗口，多任务并行处理更高效
安全隐私
• 新增通知智能隐藏，当应用接收到通知弹窗时，非本人查看手机时自动隐藏内容，确保个人隐私不被泄露
系统
全面切换为ColorOS 12.1 for OPPO，打造原汁原味的ColorOS 体验（部分 realme UI 的功能在切换为 ColorOS for OPPO 后将会丢失） • 新增注视时不息屏 • 优化长按电源按键进入电源管理界面的时长,缩短至800ms（也就是不到1秒钟的时间） • 优化温控设置
个性化
• 新增 Find 未来系列铃声 • 新增来自于多款机型的专属动态壁纸，包括 Reno6 系列、Find X2 系列、Find X3 系列、Find X5 系列 • 选取 Reno6 系列的其中一款动态壁纸作为默认壁纸
设置
• 新增关于手机马里亚纳 NPU 装饰标识 • 支持设置默认桌面（OPPO K7x） • 新增内容推荐设置入口（OPPO K7x） • 新增进程管理设置入口（OPPO K7x） • 默认开启 OTG 连接，保证使用体验 • 新增时间样式设置，支持显示秒钟、隐藏状态栏时间显示
应用分身
• 支持更多可卸载应用开启分身
浮窗
• 支持更多应用开启浮窗
控制中心、通知中心
• 去除开启开发者选项后弹出的通知提示
桌面与多任务
• 新增设置3x5~10x10桌面布局 • 去除后台锁定应用数量的限制
相机
• 新增启用哈苏相机 UI • 新增“姜文 | 灿烂”、“姜文 | 黑白”相机滤镜
应用安装程序
• 去除安装App时的安全扫描和密码验证流程
